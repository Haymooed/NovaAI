The app's environment variables haven't been injected yet. This happens when you open index.html directly from your computer.

The secrets are only injected when GitHub Actions deploys your site. To fix this:

  1. Make sure all 3 GitHub Secrets are set: SUPABASE_URL, SUPABASE_ANON_KEY, NVIDIA_API_KEY
  2. Push your files to the main branch
  3. Go to Actions → Deploy NovaAI → Run workflow
  4. Visit your live github.io URL — not the local file
